# Chapter 1: Fundamentals

## Basic Programming Model

### Exercises

1.1.1 Give the value of each of the following expressions:
   a. ( 0 + 15 ) / 2
   b. 2.0e-6 * 100000000.1
   c. true && false || true && true

Solution:
   a. 7
   b. 200.0000001
   c. true, the precedence of && is higher than ||
